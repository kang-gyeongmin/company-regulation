# Validation Checklist

작성 완료 후 반드시 아래 항목을 검토한다.

## 구조

* 편/장/절/조 구조 유지
* 조항 번호 누락 없음

## 문체

* "~하여야 한다" 사용
* "~합니다" 사용 금지

## 용어

* 사원 사용
* 직원 사용 금지
* 구성원 사용 금지

## 내용

* 목적 포함
* 적용 대상 포함
* 절차 포함
* 예외 상황 포함
* 위반 시 조치 포함
* 담당 부서 포함

## 참조

* 타 조항 참조 시 조항명 포함

예시

제16조에 따른다 ❌

제16조(구비서류 관리)에 따른다 ⭕

## 분량

* 최소 500자 이상
* 권장 700~1200자

## RAG 품질

* 조항만 읽어도 의미 파악 가능
* 지시어 사용 금지
* 모호한 표현 사용 금지
